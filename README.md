# web_crawler_trevari
 
 ### 설명
 독서클럽 trevari 멤버들의 독후감 검색사이트
 
 ### crawling link
 https://trevari.co.kr/
 
 ### user link
 https://trevari-thinkshare.tk/
 
 ### 사용 방법
 1. 검색창에 책 이름 검색 (부분 검색 가능)
 2. 일치하는 책 이름 클릭
 3. 독후감 제목 클릭 -> 트레바리 공식 홈페이지로 연결
 4. 로그인 (로그인 기록이 남아있는 경우 필요 없음)
 5. 독후감 읽기 (현재 멤버만 독후감 읽기 가능)
